# AutoUnzipper — 폴더 감시 자동 압축 해제

지정한 폴더에 **zip을 넣으면 자동으로 풀어주는** 도구입니다.
작업표시줄 **트레이 아이콘**으로 동작하고, 콘솔 창은 뜨지 않습니다.

- 감시 폴더에 zip 드롭 → `zip이름/` 폴더로 자동 해제
- 원본 zip → `_done/` 폴더로 이동
- 중첩 zip(zip 안의 zip)도 재귀 해제
- 윈도우 시작 시 **자동 실행**

> **Python 설치 필요 없음.** exe 안에 다 들어 있습니다. (Windows 전용)

---

## 🚀 설치 (3단계)

1. repo를 받습니다.
   ```
   git clone https://github.com/gobangMkt/TOMY_PUBLIC
   ```
   (이미 있으면 `git pull`)
2. `AutoUnzipper\` 폴더의 **`설치.bat`** 더블클릭
3. 감시할 폴더 입력 (그냥 Enter 누르면 **다운로드 폴더**)

끝입니다. 트레이 아이콘이 바로 뜨고, 다음 부팅부터 자동 실행됩니다.

---

## 🎛 사용법

작업표시줄 우측 **`^`(숨겨진 아이콘)** 영역에 동그란 아이콘이 생깁니다.

| 동작 | 결과 |
|---|---|
| 아이콘 색 | 🔵 파랑=대기 / 🟢 초록=처리중 / 🔴 빨강=오류 / ⚪ 회색=일시정지 |
| **우클릭 → 감시 일시정지** | zip이 떨어져도 무시 (체크 해제 시 재개) |
| **우클릭 → 로그/설정/감시 폴더 열기** | 각 폴더·파일 바로 열기 |
| **우클릭 → 부팅 시 자동 시작** | 켜고 끄기 (설치 시 기본 ON) |
| **우클릭 → 종료** | 완전 종료 |

---

## ⚙️ 동작 구조

```
감시 폴더에 zip 드롭
        │
        ▼
   파일 쓰기 완료 대기 (크기 안정화)
        │
        ▼
   zip이름/ 폴더로 해제  →  원본 zip은 _done/ 으로 이동
        │
        ▼
   안의 zip이 또 있으면 재귀 해제
        │
        ▼
   완료 토스트 알림 🔔 (오류 시 오류 알림)
```

### 폴더·파일
| 항목 | 위치 | 역할 |
|---|---|---|
| `AutoUnzipper.exe` | 이 폴더 | 본체 (트레이 앱) |
| `config.json` | 이 폴더 | 감시 폴더 설정 (설치 시 자동 생성, 개인용·커밋 안 됨) |
| `logs/` | 이 폴더 | 처리 로그 (개인용·커밋 안 됨) |
| `_done/` | 감시 폴더 안 | 해제 끝난 원본 zip 보관 |

감시 폴더를 바꾸려면 트레이 **"설정 파일 열기"** → `watch_dir` 수정 → 트레이 **종료 후 재실행**.

---

## 🗑 제거

`제거.bat` 더블클릭 → 트레이 종료 + 부팅 자동실행 삭제.
(exe·설정은 폴더에 그대로 남으니, 완전 삭제는 폴더를 지우세요.)

---

## 🛠 문제 해결

| 증상 | 해결 |
|---|---|
| 트레이 아이콘이 안 보인다 | `^` 영역 안에 숨어 있을 수 있음. 끌어내 고정하세요. 그래도 없으면 `설치.bat` 재실행 |
| zip을 넣어도 안 풀린다 | ① 트레이 아이콘이 회색(일시정지)인지 확인 ② "감시 폴더 열기"로 경로가 맞는지 확인 |
| Windows가 exe 실행을 막는다 | "추가 정보 → 실행"으로 허용 (내부 배포 exe라 서명 없음) |
| 백신이 차단한다 | PyInstaller exe 특성상 오탐 가능. 예외 등록하거나 IT에 문의 |
