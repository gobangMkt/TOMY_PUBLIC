# 아이콘 디자인 스타일 가이드

> 스타일 레퍼런스: TOSSFACE 시각 언어 (플랫·기하학·고채도)
> 단, 아이콘은 직접 제너레이트한 SVG이므로 외부 라이브러리 의존 없음.

---

## 1. 디자인 원칙

| 항목 | 규칙 |
|---|---|
| 형태 | 단순한 기하 도형 합성 (원·라운드 사각형·곡선) |
| 스타일 | 100% 플랫 — 그래디언트·그림자·입체감 사용 안 함 |
| 외곽선 | **stroke 금지** — 모든 요소는 fill 도형으로 표현 |
| 명도/채도 | 고채도 — 페일·파스텔 톤 사용 안 함 |
| 면 분리 | `#000` 또는 `#313D4C` fill 도형으로만 분리 |

**금지 목록**: 1~2px stroke outline (Lucide/Heroicons 스타일), 그라데이션 fill, 그림자, glassmorphism

---

## 2. 캔버스 & 그리드

- **viewBox**: `0 0 20 20` (art bounds 기준)
- 안전 여백: 1px (edge 잘림 방지)
- 최소 면 두께: 2px

---

## 3. 컬러 팔레트

### 시그니처
| 토큰 | HEX | 용도 |
|---|---|---|
| `--ti-yellow` | `#FFC84D` | 이모지 베이스 (별, 전구, 돋보기 등) |
| `--ti-red` | `#EF4452` | 하트, 핀, STOP |
| `--ti-orange` | `#FF9000` | 불꽃, 강조 |
| `--ti-green` | `#23B169` | 체크, 성공 |
| `--ti-blue-sky` | `#64A7FF` | 정보, 하늘 |

### 중립 (면 분리용)
| 토큰 | HEX | 용도 |
|---|---|---|
| `--ti-black` | `#000000` | 눈동자, 작은 디테일 |
| `--ti-white` | `#FFFFFF` | 하이라이트, 내부 분리 |
| `--ti-navy` | `#313D4C` | 본체 어두운 부분, 손잡이 |

### 규칙
- 한 아이콘 안에 컬러 **5종 이내**
- 흰색(`#FFF`)은 배경으로 쓰지 않음 — fill 분리 요소로만 사용

---

## 4. 형태 규칙

**코너 라디우스**
- 정사각형 컨테이너: `4.5` (≈22%)
- 직사각형 요소: `2~3`
- 버튼·도트: 완전 원 (`50%`)

**디테일 수준** (3단계 이하만 허용)
1. Major shape (본체 1~2개)
2. Sub shape (악센트 2~4개)
3. Micro accent (1~3px 점·반짝임)

---

## 5. SVG 출력 사양

```xml
<svg xmlns="http://www.w3.org/2000/svg"
     width="20" height="20"
     viewBox="0 0 20 20"
     fill="none">
  <!-- 모든 도형: fill="#XXXXXX" -->
  <!-- stroke 속성 금지 -->
</svg>
```

---

## 6. 사용 크기

| 컨텍스트 | 크기 |
|---|---|
| 본문 인라인 | `16 × 16` |
| 리스트 아이템 | `20 × 20` |
| 배너 / 카드 아이콘 | `40 ~ 48` |
| 히어로 | `64 ~ 120` |

홀수 배율(1.5×, 2.5×) 피할 것 — 벡터 엣지 블러 발생.

---

## 7. 접근성

- 아이콘 단독으로 의미 전달 시: `<svg role="img" aria-label="...">`
- 텍스트와 함께 사용 시: `aria-hidden="true"`
